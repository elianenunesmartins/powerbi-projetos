# Dashboard Financeiro - Power BI

Dashboard desenvolvido para análise de indicadores financeiros,
permitindo acompanhar recebimentos, custos, impostos e lucro.

## Indicadores analisados

- Recebimentos
- Custos
- Impostos
- Lucro
- Margem financeira

## Funcionalidades do dashboard

- Visualização da margem financeira
- Análise do lucro por mês
- Filtro interativo por instituição financeira
- Indicadores estratégicos para acompanhamento financeiro
