Dashboards desenvolvidos durante o Intensivão de Power BI 2026,
realizado pela Hashtag Treinamentos.

Projetos incluídos neste repositório:

- Dashboard Financeiro
- Dashboard RH
- Dashboard Vendas
- Dashboard Produção

Os dashboards foram desenvolvidos com foco em análise de indicadores
de negócio, construção de KPIs estratégicos e visualização de dados
utilizando o Power BI.
